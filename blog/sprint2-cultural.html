# Add HTML and CSS to your Blog Challenge

Write a small story about HTML and CSS and their relationship. You could personify elements or speak from your own personal experience. The story should capture insight about the nature of HTML and CSS.

HTML is a markup language made with the purpose of formating and sharing information via web browsers, the letters HTML is an acronym that stands for "Hyper Text Markup Language". CSS is a styling language and is also an acronym that stands for "Cascading Style Sheets" which is used to describe the presentation of an HTML document, the purpose of CSS being to seperate the content (HTML) from the presentation of that content e.g the colours, fonts, spacing and so on.

Any moments of surprise or joy?

Honestly any time something doesn't seem to work the way it's supposed to and I manage to figure it out is really fun.

If HTML and CSS were animals, which animals do you think they would be? Why?

If I had to describe HTML and CSS as animals. HTML being all about the structure of a website, I would describe it as a Beaver which spends it's time building their dam, giving it structure to serve it'spurpose. CSS being about the style/appearance of the website, I would describe it as a Peacock it's elaborate and stylized feathers while beautiful are quite impractical from a survival standpoint and don't serve much use in the way of self preservation but are extremely useful in attracting a mate, much like CSS being used to attract visitors through visual appeal.
